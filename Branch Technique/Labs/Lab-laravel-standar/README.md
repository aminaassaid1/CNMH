# CRUD LARAVEL STANDARD

## install adminlte

```
    npm install admin-lte@3.1 --save
```
```
    npm run dev
```

## create model and controller ressource and migration and Request  and seeder

 - Projects
```
    php artisan make:model Projects -mcrRs
```
 - Tasks
```
    php artisan make:model Tasks -mcrRs
```

## Run migration 

```
    php artisan migrate
```

## Run Seeder 

```
    php artisan db:seed
```
